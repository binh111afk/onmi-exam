Icon Usage and Formatting Guidelines
No Emojis or Unicode Icons: Do not use any standard emojis or Unicode symbols (e.g., 📚, 🎯, 🧠, 🔥, ⭐).

Exclusive Library: Use icons exclusively from the lucide-react library.

Explicit Imports: Every single icon must be explicitly imported from lucide-react.

Style Preference: Prioritize outline styles for consistency across the interface.

Icon Color: Apply colors matching the current active theme.

No External Assets: Do not use external PNG images or custom SVG files if a matching icon already exists within lucide-react.
